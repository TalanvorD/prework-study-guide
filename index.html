HTML - Hyper Text Markup Language

  • HTML is the language of a web broswer. It describes what information a webpage contains and how it should be displayed.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is what will be displayed in the window or tab of the browser for that particular page
  • h1 - h6 indicate headers that announce topics
  • ul is unordered list
  • li is a list item
  • img is used to reference a source to show an image
  • alt shows alternative text that can be shown with an element
  • HTML cheatsheet

CSS - Cascading Style Sheets

  • CSS is used to control the style and layout of a webpage. Colors, spacing, fonts and other cosmetic elements can be defined by a style sheet
  • Style sheets can be internal or external. Internal is contained directly within an HTML file and external is within a seperate .css file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside of an element
  • a . indicates a class selector
  • colors can be indicated in different ways semantic type (gray) or by hexadecimal (#888888)
  • sizes can be relative (%) or fixed (px)
  • CSS reference at Mozilla.org

Git

  • Git is a way to control the version of software. It allows multiple collaborators to change a repository independently while keeping it up to date.
  • touch creates a file ex. touch index.html
  • git status checks what branch we are currently on
  • git checkout -b branch-name creates a new branch and switches to it
  • git add -A stages all files in the current and higher directories that belong to that specific repository
  • commit -m "message goes here" commits the changes in the branch and adds a message describing the changes
  • git push -u origin branch-name pushes the changes in the branch to the repository
  • Useful cheatset for Git

JavaScript

  • JavaScript is a programming language mostly used on webpages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arrays can hold many variables varying data types.
  • Arrays in JavaScript are 0-indexed, meaning they start at [0], then [1], [2], etc.
  • Function are objects that can be called on to perform specific tasks
  • for is a statement that operates on a sequence of values (arrays for example)
  • JavaScript basics at Mozilla.org
