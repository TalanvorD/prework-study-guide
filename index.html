Prework Study Guide

HTML - Hyper Text Markup Language

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is what will be displayed in the window or tab of the browser for that particular page
  • h1 - h6 indicate headers that announce topics
  • ul is unordered list
  • li is a list item
  • img is used to reference a source to show an image
  • alt shows alternative text that can be shown with an element

CSS - Cascading Style Sheets

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
  • a . indicates a class selector.
  • colors can be indicated by either semantic type (ray) or by hexadecimal (#888888)
  • sizes can be relative (%) or fixed (px)

Git - Global Information Tracker or the stupid content tracker

  • git status checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
